# 📡 RadarComercial

**RadarComercial** es una web/app diseñada para ayudar a las personas a encontrar negocios de forma rápida y eficiente. En esta primera etapa, el enfoque está en el **sector de Once (Ciudad de Buenos Aires, Argentina)**, uno de los polos comerciales más importantes del país.

---

## 📌 Descripción

RadarComercial funciona como un “radar” inteligente que permite descubrir locales, proveedores y oportunidades comerciales dentro de una zona específica. Pensado tanto para compradores como emprendedores, facilita la búsqueda de negocios según rubro, ubicación y necesidades.

---

## 🎯 Objetivos

* Facilitar la búsqueda de negocios en zonas comerciales clave
* Centralizar información relevante de locales en Once
* Mejorar la experiencia de compra y abastecimiento
* Servir como base para expandirse a otras zonas y ciudades

---

## 🌆 Alcance inicial

📍 **Zona cubierta:**

* Barrio de Once
* Ciudad Autónoma de Buenos Aires (CABA)

🛍️ **Tipos de negocios:**

* Mayoristas
* Minoristas
* Proveedores
* Importadores

---

## ⚙️ Funcionalidades

* 🔎 Búsqueda de negocios por rubro
* 📍 Exploración por ubicación
* 🏪 Fichas de comercios (info básica, contacto, etc.)
* ⭐ Sistema de valoración (próximamente)
* 📊 Insights comerciales (futuro)

---

## 📊 Uso

1. Ingresar a la aplicación
2. Buscar un rubro (ej: "telas", "electrónica", "accesorios")
3. Explorar negocios disponibles en Once
4. Contactar o visitar el comercio
